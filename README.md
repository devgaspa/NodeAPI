# Project NodeAPI: API RESTful created with NodeJS.

###Resources###

* Tasks
    * GET:
        * /tasks: Find all tasks
        * /tasks/id: Find a task
    * POST:
        * /tasks: Create a task
    * PUT:
        * /tasks/id: Update a task
    * DELETE:
        * /tasks/id: Delete a task
* Users
    * GET:
        * /users/id: Find an user
    * DELETE:
        * /users/id: Delete an user
    * POST:
        * /users: Create an user
